Tailwind CSS 4 — differenze critiche da Tailwind 3:

  NON usare:
    @tailwind base;
    @tailwind components;
    @tailwind utilities;

  USARE invece:
    @import "tailwindcss";
    (singola riga, prima di tutto il resto nel CSS)

  Le variabili CSS custom in Tailwind 4 vanno nel blocco @theme:
    @theme {
      --color-rame: #B5622A;
    }
  Tailwind 4 genera automaticamente utility classes dai nomi delle variabili:
  --color-rame genera: bg-rame, text-rame, border-rame, ring-rame ecc.

  NON serve tailwind.config.js o tailwind.config.mjs.
  La configurazione avviene direttamente nel CSS tramite @theme.

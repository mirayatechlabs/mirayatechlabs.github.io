flagship project

Braindrop

Un organizer AI pensato per catturare pensieri in forma libera e restituirli come ordine operativo. Input testuali, voce e immagini entrano nello stesso flusso; il routing decide dove devono andare; il risultato torna all'utente come task, reminder e tassonomia utile.

come funziona

Un prodotto leggibile in tre strati.

La cosa importante non e “fare tante cose con l'AI”, ma farne alcune in modo chiaro e affidabile.

capture

Input liberi

Una capture box unica per testo, voce e immagini. Il primo gesto deve essere naturale, non burocratico.

Low-friction input
route

Fallback serio

Keyword router locale, poi Groq, poi Gemini. Il sistema non dipende da una sola chiamata felice.

Reliability first
return

Contesto utile

Task, cartelle, date e calendario riportano il pensiero dentro un uso reale, non solo dentro una lista.

Order and action
stack

Il progetto principale regge perche ha fondamenta reali.

La scheda progetto non e marketing astratto: deriva dal README reale di Braindrop e dalle scelte tecniche gia dichiarate.

frontend e ux
  • Next.js 16, React 19, TypeScript strict.
  • Voice system browser-native con STT e TTS.
  • PWA installabile con supporto offline.
  • Framer Motion gia presente per micro-interazioni mirate.
data e ai
  • Firestore real-time come sync cloud.
  • Routing locale + fallback Groq + fallback Gemini.
  • localStorage e IndexedDB per continuita operativa.
  • Google Calendar come layer di sync write-only.
